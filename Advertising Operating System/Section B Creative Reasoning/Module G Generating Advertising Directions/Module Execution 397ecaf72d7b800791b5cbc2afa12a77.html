Module Execution

Module Objective

Transform a single communication objective into multiple strategic advertising directions before generating concepts or visual executions.

This module ensures that creative exploration begins with communication strategy rather than visual references.

When to Run This Module

Run this module immediately after the communication objective has been defined.

Run this module before:

Concept generation

Visual brainstorming

Pinterest research

Moodboard creation

Sketching

AI image generation

Do Not Run This Module When

The communication objective is unclear.

The real brief has not been identified.

The advertisement has already entered concept development.

Visual execution has already begun.

If any of these conditions exist, return to Module F — Finding the Real Brief.

Prerequisites

Before executing this module, the following must already exist:

✓ Communication Objective

✓ Target Audience

✓ Product or Service

✓ Business Goal

✓ Brand Context (if available)

Module Input

Required:

  1. Communication Objective
  1. Product or Service
  1. Target Audience

Optional:

  1. Festival or Occasion
  1. Offer
  1. Brand Personality
  1. Competitor Information
  1. Market Context
  1. Campaign Theme

Reasoning Objective

Explore different ways of communicating the same objective without thinking about visual execution.

The objective is not to create ideas.

The objective is to discover the strongest communication direction.

Execution Rules

  1. Think strategically before thinking visually.
  1. Do not generate concepts yet.
  1. Do not search for design references.
  1. Generate multiple directions before evaluating.
  1. Do not judge directions too early.
  1. Every direction must communicate the same objective differently.
  1. Explore obvious directions before unconventional ones.

Expected Output

The module is complete only when it produces:

Multiple communication directions (minimum 10 recommended)

One selected communication direction

A clear reason for selecting that direction

A list of rejected directions (optional but recommended)

Validation Checklist

Before continuing, verify that:

□ Every direction supports the communication objective.
□ Every direction is meaningfully different.
□ The selected direction is relevant to the audience.
□ The selected direction is broad enough to explore multiple original concepts.

If any answer is "No", repeat this module.

Failure States

The module has failed if:

Only one direction was explored.

Directions are merely different headlines for the same idea.

The direction was copied from an existing advertisement.

Pinterest or other references influenced the direction before it was chosen.

Visual execution determined the communication instead of the communication determining the execution.

The selected direction is too narrow and immediately becomes an execution.

If any failure state occurs, restart the module.

Module Output

This module produces:

Required:

  • One selected communication direction.

Optional:

  • A documented set of alternative directions that may be revisited if concept generation fails.
  • These outputs become the direct input for Module H — Finding the One Big Idea.

Success Criteria

This module is considered successful when the system can confidently answer:

"This is the conversation our advertisement is going to have with the audience."

Only after that answer is clear should concept generation begin.
